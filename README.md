# Project overview

Artist Portal (ArtPort) is an art-critique-based social media platform. This platform allows users to submit their artwork, currently in the form of images, and receive structured critiques and feedback from other users. The system is designed to provide artists with meaningful insights into their work, helping them understand what aspects are effective, which areas could be improved, and how their work is perceived by others.

To access the project, please use the link below:

[art-port-three.vercel.app](https://art-port-three.vercel.app/)


# 1.0 release features:

- Improved feedback/critique forms creation, now supports multiple pre-made templates and.
- Submitting feedback to other users via forms.
- Reviewing feedback summaries of the received feedback. 
- User profile customization:
  - Uploading banners.
  - Selecting and/or changing username.
  - Updating bio. 
- Folders and artworks organization:
  - Folder creation and deletion (with inside content or with moving the content into the parent folder).
  - Moving artworks in/out of folders.
  - Bookmarking artworks from other artists into a special folder.
- Improved artists and artworks search.
- Simple inappropriate language checks for usernames, bios, artwork titles and descriptions.

# Beta release features

- Artwork upload (limited to images).
- Basic critique form creation (only the default form is supported).
- Basic search users by their username and search artwork by title or description.
- Basic user profile customization by profile picture upload. 

# Alpha release features

- Registration/login using email and password. 



# Issue Tracking System

The bug-tracking system for this project uses a hybrid approach of collecting bug reports from users via Google Form and then transferring them to the GitHub Issues system, where we can assign developers responsible for each bug/issue, mark issues as "in progress/resolved", and other tasks.

The user can access the bug report form using the following link: https://forms.gle/1gqNdWcKkygg3MrG9 

The GitHub issues used by the development team to track the known issues/bugs of the project can be found using the following link: https://github.com/ibrax2/ArtPort/issues
